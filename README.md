# FixMe

FixMe é uma aplicação desktop Electron para otimização de sistemas Windows 10/11 com foco em jogos.

## Recursos implementados (inicial)
- Aplicar plano de energia High/Ultimate Performance
- Ajustar min/max do processador via `powercfg`
- Aumentar prioridade de processos (jogos)
- Limpeza de standby list / trimming de working set

## Como rodar (desenvolvimento)
1. Instale dependências: `npm install`
2. Rode o app: `npm run dev` (requer Windows)

**Atenção:** todas as ações sensíveis pedem elevação via UAC; não há janelas de terminal expostas para o usuário.
